## Summary

CodeFlow.Persistence.Tests is an integration test project that validates the persistence layer of the CodeFlow system against a real MariaDB database. Every test class spins up an ephemeral MariaDB 11.4 container via Testcontainers, applies EF Core migrations, exercises repository and service classes from CodeFlow.Persistence, and then disposes the container. The project covers agent configuration, role-based access control, MCP server registration, workflow definitions and sagas, assistant conversations, idempotency, notification delivery, secret encryption, and authority/refusal-event recording.
