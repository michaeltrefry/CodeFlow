## Summary

CodeFlow.Persistence.Tests is an integration test project that validates the persistence layer of the CodeFlow system against a real MariaDB database. Each test class spins up an ephemeral MariaDB 11.4 container via Testcontainers, applies EF Core migrations, exercises repository and service classes (agent configs, roles, skills, MCP servers, workflows, sagas, notifications, authority snapshots, etc.), and then tears the container down. The project also contains a small number of pure unit tests for in-memory concerns such as AES-GCM encryption, workflow node value equality, and terminal-port computation.
