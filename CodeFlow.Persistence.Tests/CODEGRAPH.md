## Summary

CodeFlow.Persistence.Tests is an integration test project for the CodeFlow.Persistence layer. It uses Testcontainers to spin up real MariaDB 11.4 instances for each test fixture, runs EF Core migrations against them, and then exercises repositories, services, and domain logic (agent configs, roles, skills, MCP servers, workflows, sagas, authority/refusal events, conversations, token usage, git host settings, and artifact storage) through round-trip, CRUD, encryption, cascade, and end-to-end behavioral tests. The project also includes a small set of pure unit tests for value objects (WorkflowNode, terminal ports, AES-GCM encryption) that do not require a database.
