CodeFlow.Persistence.Tests is an integration test project that validates the persistence layer of the CodeFlow system against a real MariaDB 11.4 database. Each test class spins up an isolated containerized MariaDB instance using Testcontainers, applies EF Core migrations, and exercises repository classes, domain services, and schema correctness for entities including agent configurations, roles, skills, MCP servers, workflow graphs, workflow sagas, assistant conversations, token usage records, and git host settings.
