## Summary

CodeFlow.Persistence.Tests is an integration test project that validates the persistence layer of the CodeFlow system against a real MariaDB database. Each test class spins up an ephemeral MariaDB container via Testcontainers, applies EF Core migrations, and exercises repository classes, domain services, and schema correctness for entities such as agent configs, agent roles, MCP servers, skills, workflows, sagas, conversations, and security/encryption concerns. A small number of unit tests (for in-memory logic like AES-GCM encryption, file-system artifact storage, and workflow node value equality) run without a container.
