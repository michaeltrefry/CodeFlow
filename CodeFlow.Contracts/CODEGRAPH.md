## Summary

CodeFlow.Contracts defines the shared abstractions and data transfer types for the notification subsystem in the CodeFlow repository. It provides interfaces and result types that decouple notification dispatching, event modeling, and provider implementations from any specific orchestration or delivery mechanism. Other projects (such as CodeFlow.Orchestration) depend on these contracts to implement concrete dispatchers and providers (Slack, Email, SMS/Twilio) without coupling to one another.
