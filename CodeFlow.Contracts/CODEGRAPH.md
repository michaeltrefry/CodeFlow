CodeFlow.Contracts defines the shared abstractions and data transfer contracts for a notification system within the CodeFlow repository. It provides interfaces and types that decouple the notification dispatch pipeline (dispatchers, providers, events, messages, routes, and results) from their concrete implementations, which reside in other projects such as CodeFlow.Orchestration.
