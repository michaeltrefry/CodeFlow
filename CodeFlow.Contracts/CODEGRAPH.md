## Summary

CodeFlow.Contracts defines the shared abstractions and data transfer types for the CodeFlow notification system. It provides interfaces and result types that decouple the notification dispatching, event modeling, and provider delivery concerns, allowing other projects (such as CodeFlow.Orchestration) to implement concrete providers (Slack, Email, SMS) and consumers against a stable contract layer.
