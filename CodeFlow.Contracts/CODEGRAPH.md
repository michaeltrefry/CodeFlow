## Summary

CodeFlow.Contracts defines the shared interfaces and data types that form the notification subsystem's contract layer. It provides abstractions for dispatching notifications (INotificationDispatcher), representing notification events (INotificationEvent), and implementing notification delivery providers (INotificationProvider), along with associated result types. This project serves as the dependency boundary that decouples the orchestration layer from concrete notification implementations such as Slack, Email, and SMS.
