## Summary

CodeFlow.Contracts defines the shared abstractions and data contracts for the CodeFlow notification system. It provides interfaces and types that decouple the notification infrastructure (dispatching, routing, delivery) from concrete implementations such as Slack, Email, and SMS providers. This project serves as the contract layer consumed by both the orchestration layer and its test counterparts.
