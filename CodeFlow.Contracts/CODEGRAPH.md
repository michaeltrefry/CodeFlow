CodeFlow.Contracts defines the shared abstractions and data types for the notification subsystem within the CodeFlow repository. It provides interfaces that decouple notification dispatching, event modeling, and provider delivery from concrete implementations, which reside in other projects such as CodeFlow.Orchestration. This project acts as a contracts/abstractions layer, enabling multiple notification providers (Slack, Email, SMS) and consumers to depend on stable interfaces rather than concrete classes.
