## Summary

CodeFlow.Contracts defines the shared interfaces and data types for the notification subsystem within the CodeFlow repository. It establishes the contracts (abstractions) that external projects implement — specifically for dispatching notifications, representing notification events, and providing notification delivery via channels such as Slack, Email, and SMS. This project acts as the dependency-inversion boundary so that orchestration and consumer logic can depend on abstractions rather than concrete implementations.
