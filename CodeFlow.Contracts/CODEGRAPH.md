## Summary

CodeFlow.Contracts is a shared contracts/abstractions project that defines the core interfaces and data types used for the notification subsystem across the CodeFlow repository. It establishes the contracts that notification dispatchers, providers, and events must implement, enabling loose coupling between the orchestration layer and its concrete notification delivery mechanisms (Slack, Email, SMS).
