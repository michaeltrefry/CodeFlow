## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, database migrations, and supporting infrastructure for persisting all core domain objects including workflows, agent configurations, roles, skills, MCP servers, LLM provider settings, assistant conversations, notifications, and authority/refusal tracking. It defines the EF DbContext, all entity type configurations, a comprehensive set of repository interfaces and their implementations, secret encryption/decryption via AES-GCM, and a file-system-backed artifact store. This project serves as the single authoritative persistence boundary consumed by orchestration, API, and host projects throughout the repository.
