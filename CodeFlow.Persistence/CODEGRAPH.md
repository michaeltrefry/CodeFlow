## Summary

CodeFlow.Persistence is the data access and persistence layer for the CodeFlow system. It defines the Entity Framework Core DbContext, all database entity models, EF entity configurations, repository implementations, and database migrations. It also provides interfaces consumed by other projects (orchestration, API) for reading and writing workflows, agent configurations, roles, skills, MCP servers, conversations, artifacts, notifications, and LLM/Git/web-search provider settings. The project additionally implements secret protection (AES-GCM encryption), a file-system artifact store, and authority/role-resolution services.
