## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow repository, providing Entity Framework Core-based repositories, entity definitions, database migrations, and supporting infrastructure for all persistent state in the system. It manages storage and retrieval for workflows, agent configurations, agent roles, MCP (Model Context Protocol) servers, LLM provider settings, assistant conversations, artifacts, token usage records, and authority/refusal tracking. It also implements cross-cutting concerns such as secret encryption (AES-GCM), a file-system artifact store, role resolution (mapping agent keys to resolved tool grants), and system data seeding.
