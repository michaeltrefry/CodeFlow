## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, EF migrations, and supporting infrastructure for persisting all application state. It defines repository interfaces and their EF-backed implementations for workflows, agent configurations, roles, MCP servers, LLM provider settings, assistant conversations, notifications, token usage records, and authority/refusal events, as well as a file-system artifact store and AES-GCM secret protection. This project serves as the single source of truth for database schema (via migrations) and the primary integration point between the CodeFlow runtime/orchestration layers and the underlying SQL database.
