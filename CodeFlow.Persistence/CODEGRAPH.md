## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow repository, providing Entity Framework Core-backed repositories, entity configurations, database migrations, and supporting services for all persistent data in the system. It manages storage for workflows, agents, roles, LLM provider settings, MCP servers, assistant conversations, notifications, artifacts, and authority/refusal tracking. The project also includes file-system artifact storage, AES-GCM secret encryption, and a role-resolution service that bridges runtime agent tool grants with the database.
