## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, database migrations, and supporting infrastructure for all persistent state. It manages storage for workflows, agent configurations, roles, MCP servers, skills, LLM provider settings, assistant conversations, token usage records, artifact files, and an authority/refusal event subsystem. This project acts as the shared persistence contract for other projects in the repository (API, Orchestration, etc.) through well-defined repository interfaces and injectable implementations.
