## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, database migrations, and domain model mappings. It defines interfaces and implementations for storing and retrieving agents, workflows, roles, MCP servers, LLM provider settings, conversation history, token usage, and file-based artifacts, while also handling secret encryption and role/tool resolution. This project serves as the shared persistence contract consumed by orchestration, API, and host projects throughout the repository.
