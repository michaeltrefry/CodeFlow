## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, database migrations, and supporting infrastructure for persisting all core domain entities including workflows, agent configurations, roles, skills, MCP servers, LLM provider settings, prompt partials, artifact storage, and saga state. It defines repository interfaces consumed by orchestration and API layers, implements them against a relational database (via CodeFlowDbContext), and also provides a file-system-based artifact store and AES-GCM secret encryption for sensitive configuration values.
