## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based persistence for all domain entities including workflows, agents, MCP servers, LLM provider settings, assistant conversations, notifications, and authority/role management. It defines repository interfaces and their EF Core implementations, manages database migrations, handles secret encryption/decryption via AES-GCM, and provides a file system artifact store for binary content. This project serves as the shared persistence contract and implementation used by the API, orchestration, and runtime layers throughout the repository.
