## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow repository, providing Entity Framework Core-based persistence for all domain entities including workflows, agent configurations, roles, MCP servers, LLM provider settings, assistant conversations, and authority/refusal tracking. It defines repository interfaces and their EF Core implementations, manages database migrations, handles secret encryption via AES-GCM, and provides a file-system-backed artifact store. This project serves as the shared persistence contract consumed by the API, orchestration, and test projects throughout the repository.
