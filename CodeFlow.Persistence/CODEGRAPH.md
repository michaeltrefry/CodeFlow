## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, database migrations, and supporting services for all persistent state in the application. It manages storage for workflows, agents, roles, MCP servers, LLM provider settings, assistant conversations, notifications, token usage, and authority/security records. The project also includes a file-system artifact store, AES-GCM secret encryption, role resolution logic, and seeding utilities, serving as the single authoritative persistence boundary consumed by the API and orchestration layers.
