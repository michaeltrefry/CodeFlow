CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity configurations, and database migrations for all application domains including workflows, agent configurations, MCP servers, LLM provider settings, assistant conversations, notifications, token usage tracking, and authority/role enforcement.
