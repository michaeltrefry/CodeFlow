## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, database entity definitions, EF migrations, and supporting infrastructure for persisting workflows, agents, roles, MCP servers, LLM provider settings, conversation history, and artifact storage. It defines the repository interfaces consumed by the orchestration and API layers, implements those interfaces against a relational database (with encrypted secrets via AES-GCM), and provides a file-system-backed artifact store for binary content. The project also includes seeders for system-managed data and a role resolution service that assembles tool grants for agents.
