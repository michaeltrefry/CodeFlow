## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, implementing Entity Framework Core repositories, database entities, migrations, and storage abstractions. It provides typed repository interfaces and their concrete implementations for all major domain objects including workflows, agents, roles, conversations, notifications, and authority/security records, along with a file system artifact store and AES-GCM secret encryption. The project serves as the single source of truth for database schema evolution (via EF migrations) and bridges the runtime domain models with persistent storage.
