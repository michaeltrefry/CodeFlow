## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, database migrations, and supporting infrastructure for persisting all core domain objects including workflows, agent configurations, roles, MCP servers, LLM provider settings, prompt partials, skills, and workflow saga state. It also includes a file-system-backed artifact store, AES-GCM secret encryption, and a role resolution service that maps agent keys to resolved tools and skills. This project serves as the shared persistence contract (via interfaces) consumed by orchestration, API, and test projects throughout the repository.
