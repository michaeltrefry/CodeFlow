## Summary

CodeFlow.Persistence is the data access layer for the CodeFlow system, providing Entity Framework Core-based repositories, entity definitions, EF migrations, and supporting infrastructure for persisting all domain objects: workflows, agent configurations, roles, skills, MCP servers, assistant conversations, artifacts, notifications, token usage, LLM/git/web-search provider settings, and authority/refusal audit trails. It defines repository interfaces consumed by the API and orchestration layers, implements them against a single SQL database (CodeFlowDbContext), and provides supplementary services such as AES-GCM secret encryption, file-system artifact storage, and role/tool resolution for agent authorization.
