## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform — a system for authoring, running, and observing AI-agent workflows. It exposes HTTP endpoints for managing workflows, agents, agent roles, traces (workflow execution runs), an AI assistant chat interface, MCP (Model Context Protocol) server integrations, notification routing, and package import/export.
