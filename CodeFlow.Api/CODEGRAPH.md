## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, which orchestrates AI agent workflows. It exposes HTTP endpoints for managing workflows, agents, agent roles, traces (workflow execution runs), MCP servers, notifications, and an AI-powered assistant. The project also handles authentication/authorization, workflow package import/export, workflow validation, trace evidence bundles, and an idempotent SSE-based chat interface backed by OpenAI and Anthropic LLM providers.
