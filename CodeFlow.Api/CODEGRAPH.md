## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration system that manages AI agents, workflow definitions, execution traces, and associated tooling. It exposes HTTP endpoints for creating and managing workflows, agents, roles, MCP servers, skills, and traces, while also providing an AI-powered assistant (supporting both OpenAI and Anthropic providers) with tool-calling capabilities. The project handles authentication/authorization, workflow validation, package import/export, cascade version bumping, and real-time trace event streaming via SSE.
