## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration and AI-agent management system. It exposes HTTP endpoints for managing workflows, agents, agent roles, MCP servers, traces, and an LLM-backed assistant, while also handling authentication, validation, package import/export, and real-time trace event streaming. The project wires together persistence repositories, runtime services, and an AI assistant (supporting both OpenAI and Anthropic providers) into a cohesive REST API surface.
