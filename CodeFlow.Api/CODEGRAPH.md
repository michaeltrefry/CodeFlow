## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform — a system for orchestrating AI agent workflows. It exposes REST endpoints for managing workflows, agents, traces (workflow execution runs), human-in-the-loop (HITL) decisions, assistant conversations, and package import/export. It also hosts an AI assistant (supporting OpenAI and Anthropic backends) with a rich tool system, idempotency infrastructure for streaming turns, workflow validation pipelines, authentication/authorization, and event brokering via MassTransit.
