## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration system that manages AI agents, workflows, traces, and an LLM-powered assistant. It exposes REST endpoints for managing agents, workflows, MCP servers, skills, traces, notifications, and replay operations, while also hosting a streaming AI assistant (supporting both OpenAI and Anthropic providers) equipped with dozens of tool integrations for querying and manipulating system resources. The project also includes workflow validation pipelines, package import/export, template materialization, authorization, and idempotency infrastructure.
