## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration and AI agent management system. It exposes HTTP endpoints for managing agents, workflows, traces, MCP servers, skills, and roles, while integrating an AI assistant (supporting both OpenAI and Anthropic backends) with tool-use capabilities. The project handles authentication/authorization, workflow validation, package import/export, template materialization, and real-time trace event streaming.
