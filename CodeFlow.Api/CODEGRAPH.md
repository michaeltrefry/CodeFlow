## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a system for authoring, executing, and monitoring AI-agent-driven workflows. It exposes HTTP endpoints for managing workflows, agents, traces (workflow execution runs), human-in-the-loop (HITL) decisions, MCP servers, and an AI assistant chat interface. It also owns the package import/export system for distributing workflow and agent definitions, a multi-rule validation pipeline for workflow authoring, and an idempotent SSE streaming layer for the assistant.
