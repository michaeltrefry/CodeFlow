## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a system for defining, executing, and managing AI agent workflows. It exposes REST endpoints for managing agents, workflows, traces, MCP (Model Context Protocol) servers, skills, agent roles, and an AI-powered assistant chat interface. The project also handles authentication/authorization, workflow validation, package import/export, cascade version bumping, and real-time trace event streaming.
