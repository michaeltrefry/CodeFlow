## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a system for authoring, executing, and monitoring AI agent workflows. It exposes HTTP endpoints for managing agents, workflows, traces, MCP servers, skills, and roles, and integrates an AI assistant (supporting both OpenAI and Anthropic) that can answer questions about and manipulate the system via tool calls. It also handles authentication/authorization, workflow validation, package import/export, cascade version bumping, and real-time trace event streaming.
