## Summary

CodeFlow.Api is the main ASP.NET Core web API layer for the CodeFlow platform, a workflow automation and orchestration system. It exposes HTTP endpoints for managing workflows, agents, agent roles, traces, MCP servers, skills, notifications, and a built-in AI assistant. The project handles authentication/authorization, workflow validation, package import/export, template materialization, trace evidence bundling, real-time trace event streaming, and integrates with both OpenAI and Anthropic LLM providers for an AI assistant that can introspect and manipulate the platform's resources via tool calls.
