## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a system for orchestrating AI agent workflows. It exposes HTTP endpoints for managing workflows, agents, traces (workflow execution runs), an AI assistant chatbot, and supporting resources such as MCP servers, notification providers, and LLM provider settings. It acts as the primary entry point for clients, delegating to persistence, runtime, and orchestration projects while enforcing authentication, authorization, validation, and idempotency concerns.
