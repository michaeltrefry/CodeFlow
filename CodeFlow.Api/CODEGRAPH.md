## Summary

CodeFlow.Api is the ASP.NET Core Web API layer for the CodeFlow platform — a workflow orchestration and AI agent management system. It also implements authentication/authorization, workflow validation pipelines, package import/export, workflow templates, and event-driven trace observation via a message bus.
