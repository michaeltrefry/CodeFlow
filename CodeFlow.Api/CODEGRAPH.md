CodeFlow.Api is the ASP.NET Core Web API layer for the CodeFlow platform, a workflow orchestration system that manages AI agent-driven workflows. The project handles authentication/authorization, workflow validation, package import/export, trace evidence bundles, HITL (Human-In-The-Loop) decisions, and an LLM-backed assistant with tool-calling capabilities against OpenAI and Anthropic providers.
