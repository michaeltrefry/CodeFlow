## Summary

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration and AI-agent management system. It exposes REST endpoints for managing workflows, agents, roles, traces (execution runs), MCP servers, and an AI assistant; it also handles authentication, authorization, package import/export, workflow validation, and real-time trace event streaming. The project coordinates between a persistence layer, a runtime execution layer, and external LLM providers (OpenAI, Anthropic) to provide a complete control plane for authoring and operating AI-driven workflows.
