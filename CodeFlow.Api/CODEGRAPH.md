## Summary

CodeFlow.Api is an ASP.NET Core web API layer that exposes HTTP endpoints for managing AI agent workflows, including agent configuration, workflow authoring and versioning, execution tracing, package import/export, and an LLM-backed assistant chat interface. It handles authentication/authorization, validation pipelines, message-bus consumers for trace events, and serves as the primary integration surface between the persistence layer, runtime orchestration, and front-end clients. The project also includes a tool-calling assistant system that allows an AI assistant to query and manipulate workflows, agents, traces, and roles through structured tool invocations via OpenAI and Anthropic providers.
