## Summary

CodeFlow.Api is the HTTP API layer of the CodeFlow platform, a workflow orchestration system that manages AI agents, workflows, traces, and human-in-the-loop (HITL) processes. It exposes REST endpoints for creating and managing workflows, agents, agent roles, MCP servers, LLM provider settings, notifications, and trace execution/replay, while also hosting an AI assistant (supporting OpenAI and Anthropic) with tool-calling capabilities that allow the assistant to inspect and manipulate the system. The project also handles authentication/authorization, workflow validation, package import/export, and real-time trace event streaming.
