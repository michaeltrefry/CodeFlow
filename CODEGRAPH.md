### CodeFlow

CodeFlow is a full-stack workflow orchestration platform with an Angular frontend (codeflow-ui) and a .NET backend. The system enables users to design, execute, and monitor AI agent-driven workflows through a visual canvas editor, manage LLM providers, trace execution history, handle human-in-the-loop decisions, and configure notifications. The UI provides rich editing capabilities including node-based workflow design, agent configuration, script editing with Monaco, replay/debugging tools, and an integrated assistant chat interface.

### CodeFlow.Api

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration and AI-agent management system. It exposes HTTP endpoints for managing workflows, agents, agent roles, MCP servers, traces, and an LLM-backed assistant, while also handling authentication, validation, package import/export, and real-time trace event streaming. The project wires together persistence repositories, runtime services, and an AI assistant (supporting both OpenAI and Anthropic providers) into a cohesive REST API surface.

CodeFlow.Orchestration is the core workflow execution engine for the CodeFlow system.
