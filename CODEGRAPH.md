## Summary

CodeFlow is a full-stack .NET and Angular application for orchestrating multi-agent AI workflows. The backend is built around a message-driven pipeline using MassTransit and RabbitMQ, where a saga state machine tracks execution state as agents are invoked sequentially according to configurable workflow graphs. A REST API layer exposes endpoints for managing agents, workflows, and execution traces, including dead-letter queue operations and server-sent event (SSE) streaming for real-time trace monitoring, with role-based authorization controlling access.
