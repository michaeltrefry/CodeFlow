CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a workflow orchestration and AI agent management system. It exposes HTTP endpoints for managing workflows, agents, agent roles, MCP servers, traces, notifications, and an AI assistant chat interface. The project wires together persistence, runtime, and orchestration concerns through dependency injection, providing authentication/authorization, workflow validation pipelines, package import/export, template materialization, and an LLM-backed assistant with tool-calling capabilities.
