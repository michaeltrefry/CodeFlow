## Summary

CodeFlow is a full-stack AI agent workflow orchestration platform consisting of an Angular frontend (codeflow-ui) and a layered .NET backend. The system allows users to compose, version, and execute pipelines of LLM-backed agents through a visual node-based workflow editor. The backend is structured into well-defined layers: CodeFlow.Api (ASP.NET Core REST API exposing endpoints for workflows, agents, traces, MCP servers, notifications, and an AI assistant), CodeFlow.Orchestration (saga-based workflow execution engine built on MassTransit, handling agent sequencing, retry logic, swarm coordination, human-in-the-loop decisions, dry-run simulation, and replay), CodeFlow.Runtime (core agent invocation engine supporting Anthropic, OpenAI, and LM Studio providers, MCP protocol tool execution, Git workspace management, Scriban template rendering, and authority/admission enforcement), CodeFlow.Persistence (Entity Framework Core data access layer targeting MariaDB, with AES-GCM secret encryption, artifact storage, and repository implementations for all domain entities), CodeFlow.Host (composition root/infrastructure layer wiring RabbitMQ messaging, LLM provider configuration, MCP server connections, VCS workspace management, and observability into the ASP.NET host via DI extension methods), CodeFlow.Contracts (shared messaging abstractions and interfaces for the notification subsystem), and CodeFlow.Worker (background worker service for asynchronous processing). Architectural boundaries are enforced by CodeFlow.Architecture.Tests using static source scanning. The test surface covers integration tests (real MariaDB and RabbitMQ via Testcontainers), unit tests with fakes/stubs, and architecture compliance checks. Externally, CodeFlow makes HTTP calls to a CodeGraph service and exchanges messages bidirectionally with a codeflow-team system via a publish/subscribe channel. Key third-party dependencies include MassTransit (saga/message bus), Entity Framework Core (MariaDB), Testcontainers (integration testing), Scriban (templating), and SDKs for Anthropic, OpenAI, and LM Studio. The platform supports features including HITL review loops, swarm agent patterns, dataflow scripting nodes, MCP server integration, role-based access control, package import/export, workflow versioning, and real-time trace event streaming.
