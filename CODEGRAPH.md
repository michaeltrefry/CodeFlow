### CodeFlow

CodeFlow is a full-stack workflow orchestration platform with an Angular-based UI (codeflow-ui) and a .NET backend. The UI provides a visual workflow editor, agent configuration, trace inspection, LLM provider management, and an AI assistant interface. It enables users to design, execute, and monitor multi-agent AI workflows with features like dry runs, replay, HITL (human-in-the-loop) decisions, MCP server integration, and package import/export.

### CodeFlow.Api

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform, a system for defining, executing, and managing AI agent workflows. It exposes REST endpoints for managing agents, workflows, traces, MCP (Model Context Protocol) servers, skills, agent roles, and an AI-powered assistant chat interface. The project also handles authentication/authorization, workflow validation, package import/export, cascade version bumping, and real-time trace event streaming.

CodeFlow.Api.Tests is the automated test project for the CodeFlow API, containing unit tests, integration tests, and end-to-end tests. It exercises the full API surface including workflow management, agent configuration, assistant tooling, validation pipelines, dead-letter queue handling, MCP server integration, trace/replay functionality, and workflow template materialization. The project uses Testcontainers to spin up real MariaDB and RabbitMQ instances for integration and end-to-end test scenarios, and xUnit as the test framework.

CodeFlow.Host is an infrastructure and composition root project that wires together runtime dependencies for the CodeFlow system. It acts as the host-layer glue between the runtime domain abstractions (CodeFlow.Runtime) and concrete infrastructure implementations (persistence, transport, external AI providers).

CodeFlow.Orchestration is the workflow execution engine of the CodeFlow repository. It also provides a dry-run simulation engine for previewing workflow execution paths without invoking live agents, plus scripting, dataflow analysis, replay, and token-usage tracking infrastructure.
