## Summary

CodeFlow is a multi-component platform for orchestrating multi-agent AI workflows, built on .NET (backend) and Angular (frontend). The backend exposes a REST API that serves as the entry point for managing agents, workflows, traces, MCP (Model Context Protocol) servers, and agent roles. Workflow execution is driven by a message-based orchestration engine that uses MassTransit sagas over RabbitMQ, enabling stateful, event-driven coordination of multi-step agent pipelines. Persistence is handled via MariaDB/MySQL. The agent runtime integrates with multiple AI model providers (OpenAI, Anthropic, LM Studio) and supports MCP tool integration, allowing agents to invoke external tools within workflows. Human-in-the-loop (HITL) task management enables workflows to pause and await human input before continuing. A role-based tool access control system governs which tools each agent role may use. The Angular frontend (codeflow-ui) communicates with the REST API to provide a UI for all of these capabilities. The system has no observed external dependents; it depends on RabbitMQ for messaging, MariaDB/MySQL for storage, and third-party AI provider APIs for model inference.
