### CodeFlow

CodeFlow is a full-stack application for orchestrating AI agent workflows. It consists of an Angular frontend (codeflow-ui) that provides a visual workflow editor, agent management, trace/replay inspection, chat assistant interface, and settings administration, backed by a .NET orchestration and persistence layer that executes multi-agent workflows, manages LLM providers, handles human-in-the-loop decisions, and delivers notifications. The project enables users to design, run, monitor, and replay workflows composed of AI agents with support for subflows, swarm protocols, review loops, MCP tool servers, and role-based access control.

### CodeFlow.Api

CodeFlow.Api is the ASP.NET Core Web API layer for the CodeFlow platform, which orchestrates AI agent workflows. It exposes REST endpoints for managing agents, workflows, traces, MCP servers, agent roles, skills, notifications, and a conversational AI assistant. It also contains the authorization layer, workflow validation pipeline, workflow package import/export, workflow templates, and real-time trace event streaming.

Its role is to decouple the notification API surface from any concrete implementation.

It provides DI registration extensions, configuration resolution for LLM providers (LMStudio, Anthropic, OpenAI), RabbitMQ-based message bus and dead-letter queue management, workspace lifecycle services, MCP server connection provisioning, and OpenTelemetry observability setup. ### CodeFlow.Orchestration

CodeFlow.Orchestration is the central workflow execution engine for the CodeFlow system. It orchestrates multi-step AI agent workflows using a MassTransit saga state machine, handling agent invocation, decision routing, retry logic, subflow/review-loop execution, swarm coordination, human-in-the-loop (HITL) task management, and notification delivery. It also provides dry-run simulation, replay/drift detection, scripting (Scriban templates and Jint JS), and token usage tracking.
