The backend handles workflow execution, agent invocation, persistence, authentication, and notification delivery.

### CodeFlow.Api

CodeFlow.Api is the main ASP.NET Core API layer for the CodeFlow platform — a system for defining, executing, and managing AI-driven workflows composed of agent nodes. It exposes HTTP endpoints for managing workflows, agents, agent roles, MCP servers, skills, traces, and an AI assistant chatbot; implements authorization, workflow validation, package import/export, trace evidence bundling, and workflow template materialization; and wires up a message bus for consuming agent invocation events.

CodeFlow.Orchestration is the workflow execution engine for the CodeFlow system. It acts as the central orchestration layer between persisted workflow definitions, agent runtime invocers, and downstream consumers.
