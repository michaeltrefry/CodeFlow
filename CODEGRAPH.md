### CodeFlow

CodeFlow is a full-stack workflow automation platform with an Angular frontend (codeflow-ui) and a .NET backend. The system enables users to design, execute, and monitor AI agent-driven workflows through a visual canvas editor, manage LLM provider configurations, handle human-in-the-loop (HITL) decisions, track execution traces with token usage, and configure notification routing. The frontend provides API service layers, UI components, and a rich workflow editor with versioning, dry-run testing, and package import/export capabilities.

### CodeFlow.Api

CodeFlow.Api is the ASP.NET Core web API layer for the CodeFlow platform — a workflow orchestration system that coordinates AI agents through configurable multi-step workflows. It exposes HTTP endpoints for managing workflows, agents, traces (workflow execution runs), an AI assistant interface, and supporting resources like MCP servers, skills, and notification providers. The project wires together persistence, runtime execution, authorization, and an LLM-backed assistant with tool-calling capabilities into a cohesive API surface.
