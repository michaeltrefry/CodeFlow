# CodeFlow

CodeFlow is an event-driven multi-agent workflow system with a provider-neutral runtime, bus-backed orchestration, versioned configuration, and a management UI.

The repository is at kickoff stage. The implementation reference lives here:

- [Kickoff implementation plan](./docs/kickoff-implementation-plan.md)
- [Local integration stack](./docs/local-integration-stack.md)

The `CodeFlow` Kanban project and `Kickoff` epic remain the operational source of truth for execution status and work slicing.
