## Summary

CodeFlow.Api.Tests is the test project for the CodeFlow API, containing unit, integration, and end-to-end tests that validate the API's behavior across all major subsystems including workflow management, agent configuration, assistant tools, validation pipelines, dead-letter queue handling, MCP server management, notification delivery, trace/replay functionality, and workflow package import/export. It uses Testcontainers to spin up real MariaDB and RabbitMQ instances for integration tests, and a WebApplicationFactory-based test host (CodeFlowApiFactory) for HTTP-level integration tests against the full ASP.NET Core application stack.
