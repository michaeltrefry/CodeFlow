CodeFlow.Api.Tests is the test project for the CodeFlow API, containing integration tests, unit tests, and end-to-end tests covering the API's major subsystems: workflow management, agent configuration, template materialization, assistant tools, authentication/authorization, dead-letter queue handling, validation pipelines, and package import/export.
