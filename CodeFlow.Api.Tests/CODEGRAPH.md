## Summary

CodeFlow.Api.Tests is the automated test suite for the CodeFlow API, containing integration tests, unit tests, and end-to-end tests covering all major API subsystems including workflow management, agent configuration, assistant tooling, validation pipelines, dead-letter queue handling, notification endpoints, trace management, and workflow package import/export. It uses Testcontainers to spin up real MariaDB and RabbitMQ instances for integration scenarios, and Microsoft's WebApplicationFactory for full HTTP-level integration tests against a hosted version of the API. The project exercises both the HTTP layer (endpoint tests) and internal service logic (unit/component tests) with extensive use of fakes, stubs, and recording test doubles.
