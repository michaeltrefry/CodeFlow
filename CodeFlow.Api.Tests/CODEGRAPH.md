## Summary

CodeFlow.Api.Tests is the test project for the CodeFlow API, containing integration, unit, and end-to-end tests covering all major subsystems: workflow authoring and validation, agent configuration, assistant/LLM tooling, template materialization, dead-letter queue handling, authentication, notifications, trace replay, and workflow package import/export. It uses Testcontainers to spin up real MariaDB and RabbitMQ instances for integration tests, and a WebApplicationFactory-based test host (CodeFlowApiFactory) to exercise HTTP endpoints. The project spans both fast in-process unit tests (with fakes/stubs) and slower end-to-end tests that exercise the full stack against containerized infrastructure.
