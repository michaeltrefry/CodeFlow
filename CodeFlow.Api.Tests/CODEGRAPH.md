## Summary

CodeFlow.Api.Tests is the integration and unit test project for the CodeFlow API, covering HTTP endpoints, domain logic, validation rules, assistant tools, workflow templates, dead-letter queue handling, and package import/export. It uses Testcontainers to spin up real MariaDB and RabbitMQ instances for end-to-end tests, and provides a shared WebApplicationFactory (CodeFlowApiFactory) that wires those containers into the ASP.NET Core test host. The project exercises every major subsystem of the API — agents, workflows, traces, notifications, auth, MCP servers, and the LLM assistant — through a combination of full HTTP round-trips, service-layer tests with in-memory databases, and pure unit tests with stub/fake collaborators.
